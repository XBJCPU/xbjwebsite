许比嘉个人作品集网站
====================

这是一套完整的单页静态网站，无需安装程序或构建工具。

本地预览
--------
直接双击 index.html 即可浏览。

发布到互联网
------------
将 portfolio-site 整个文件夹上传到任意静态网站托管平台即可。
请保留 index.html、styles.css、script.js 和 assets 文件夹的相对位置。

页面结构
--------
1. 首页
2. 代表作品
3. 项目经历
4. 商业合作
5. 内容方法论
6. 联系方式

主要功能
--------
1. 完整 Opening Animation、首屏标题遮罩进场与背景视频。
2. 固定悬浮导航及当前章节状态提示。
3. 滚动章节标题进场、卡片 stagger、图片 reveal 和轻量视差。
4. 默认展示 AI 动画，可切换游戏杂谈与动画杂谈。
5. 作品卡片可跳转对应视频。
6. 可直接下载新版简历 PDF 与完整作品集 PDF。
7. 电脑端提供定制像素星芒鼠标与点击波纹反馈。
8. 默认循环播放背景音乐，可通过顶部音乐按钮暂停与继续。
9. 全站正文、标签与数据字号不低于 18px，并适配电脑、平板和手机。
10. 支持减少动态效果与节省流量系统设置。
11. 不依赖外部框架或在线脚本，上传后即可运行。

主要文件
--------
index.html                       网站内容
styles.css                      视觉与响应式样式
script.js                       开场、滚动、导航与分类交互
assets/hero-background.mp4     首屏背景视频
assets/cyan-hotwire.mp3        循环背景音乐
assets/xu-bijia-resume.pdf      个人简历
assets/xu-bijia-portfolio.pdf   完整作品集
